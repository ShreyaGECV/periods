Create Table

Write a SQL query to create a table named "book_records" with columns like:

  • Book_Library_No
  • Book_Name
  • Author_Name
  • Book_Edition
  • Price
CREATE TABLE book_records (
  Book_Library_No INT PRIMARY KEY,
  Book_Name VARCHAR(100),
  Author_Name VARCHAR(100),
  Book_Edition VARCHAR(50),
  Price DECIMAL(10,2)
);
